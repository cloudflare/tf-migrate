# Security Policy

## Reporting Security Vulnerabilities

If you believe you have found a security vulnerability in tf-migrate, please report it responsibly. **Do not open a public GitHub issue for security vulnerabilities.**

Please report vulnerabilities through [Cloudflare's vulnerability disclosure process](https://www.cloudflare.com/.well-known/security.txt).

We appreciate your help in keeping tf-migrate and the Cloudflare ecosystem secure.
